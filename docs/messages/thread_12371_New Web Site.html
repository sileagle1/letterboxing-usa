> least the "old" main web site. That site
> (http://www.letterboxing.org) did not have functionality for people
> to enter their own clues and could only be updated by web masters
> making changes to the web pages. So when people placed a new
> letterbox, they used this talk list as a way to announce that and the
> web masters picked it up from there and added it to the "main site."
>
> The "new" main site (which is temporarily found at
> http://letterboxing.wesgarrison.us/) has the functionality for people
> to just add their own clues, rather than having to have the web
> masters do all of the work. It also has the ability to "announce"
> the box to a special yahoo groups list that anyone can join, and
> therefore get notification of new boxes.
>
> For a while, the two "main sites" are co-existing to allow people
> time to get their clues added to the new site and still leaving them
> available on the old site so people can find them to go out
> letterboxing. After the transition is completed and we are using the
> new site exclusively, there will not be a need to post new clues to
> this list any longer.
