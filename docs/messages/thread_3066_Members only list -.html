Well, that's the beauty of Yahoogroups -- there are different degrees of
"members only"...most of my other lists are set up so that anyone can read
messages (and archived messages, too), but in order for someone to post to
the list, they have to "subscribe" to the list. By 'subscribe' I don't mean
that the moderator has to approve membership (though that is possible, too,
I don't think it necessary here) -- it's an instant kind of thing. Then
there is virtually no difference in the list than how it is now, except that
if someone does start flooding the list with spam, the moderator can remove
them.

If there was someone who *really* didn't want to register with Yahoogroups
(which I believe you must do anyway to read the messages currently, no?) and
wanted to post clues, I'm sure that some kind soul would step up and would
post that clue to the list for them. It could easily be another email
address on the website if necessary.

Just my .02,
Ann in MA
