From: Silent Doug (silentdoug@letterboxing.info) | Date: 2006-05-07 21:56:32 UTC-04:00
At 07:38 PM 5/7/2006, you wrote:
>I'm thinking about posting clues that would only be photos without
>words. Can someone tell me the names of some boxes which have photo
>clues? How do you add photos? -Rose

One way to do this is to use a photo hosting service like
tinypic.com. You can upload an image to their web site, and they'll
host it on their web site for you. They'll give you a link to your
image on their web site, and you can use that link to embed the image
in a web page, or just send the link to people or post it online. The
URL will have some random filename like this, though:
http://i3.tinypic.com/xfe2bn.jpg

It's pretty sweet -- a free service that will save you from mountains
of heart-breaking work that you might otherwise have to do to build
your own web site....

From: TJ_Mich (tj_mich@ameritech.net) | Date: 2006-05-08 05:39:32 UTC-07:00
If you want a couple more examples... we have a box like this, too: "The Sun Shines" in Michigan. Another one is SpringChick's "Copper Country Autumn," also in Michigan.

http://letterboxing.org/BoxView.php?boxnum=14843&boxname=The_Sun_Shines

If you don't have your own website, I second Silent Doug's idea for the photo hosting service. In our case, we just loaded the photos to our GeoCities site, which is probably a slight bit more complicated, but works basically the same way. Anyway, this is all possible due to the website enhancements (a while back) that allow everyone to include HTML tags in our clues... a great feature! Thanks to whoever did that!

--TJ_Mich
