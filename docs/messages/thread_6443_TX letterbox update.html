Fellow letterboxers & esp those interested in the Austin area,
There have been some postings on TX letterboxes so I thought I'd give
an update. Also, please see my earlier posting #12599 for an update
on the Brownsville boxes or type "TX letterbox" in the search engine.

Lost Pines Letterbox, Buescher State Park, Smithville TX: very kid
friendly because there is a shortcut provided in the clues; box is in
great condition; beautiful hand-carved stamp.

O-Knots Letterbox Series, Bastrop State Park, Bastrop, TX: a little
more challenging because you use an orienteering map; a lot of fun
and a lot of poison ivy; kid friendly; not hand-carved, but beautiful
celtic knots; found knots 1, 2 & 4; kids too tired for #3; all boxes
in great shape; knot #1 was out in the open probably from heavy
rains; we replanted it where it was supposed to be.

Shavian Cowboy Letterbox: very kid friendly, but an adult needs to
decipher the clues which are written in code; box in great shape;
great hand-carved stamp; the clues are not currently found on the
letterboxing.org website but if you go the clues for the above boxes,
the owner has a link to his own website which has more TX boxes.

Mt. Bonnell Letterbox, Austin: our P2 and we thank Charles (and
family) for appreciating our hard work and giving us our first
compliments. We dried it out twice already due to the heavy rains and
doubled the plastic bags.

Barton Springs Greenbelt Letterbox, Austin: our P3 and anxious for
visitors; very kid friendly except for some thorny vines.

Wraith of the Woods Letterbox,Austin: we did not visit because you
need a GPS

Gordian Knot Letterbox, Austin: we did not visit because you need a
GPS.

Enchanted Rock Letterbox, Fredericksburg, TX: we did not visit yet.

Hope this posting helps esp Rosana, Charles, Gwen & Don!
