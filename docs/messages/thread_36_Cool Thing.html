I found this cool new software called ICQ.
http://www.pclink.com/elf/icqsetup.exe

It monitors the web to see if others are online.
If they are they show up on your list and you
can then send live messages back and forth.
This would be useful if we wanted to schedule
a discussion for instance. Have you ever
wanted to do that? I have.

Anyway, the setup program can be downloaded
by clicking the above link or by going to the
Kids Corner and clicking on the little boy's
right heel.

Let me know what you think or if you think
you want to try it. I think it would be really
cool. So far I only have my son and my sister
on my list, but I think I will add all of you
so that if you join I will see you and can chat
with you about letterboxing.

Dan'l
