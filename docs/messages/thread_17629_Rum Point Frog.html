Just wanted to let you know that we did indeed find the Rum Point Frog
letterbox when we were on Grand Cayman Island for vacation. It was in
fine shape- a bit well hidden- we had to dig a bit for it probably from
hurricane debris and wade through some dense vegetation to get to the
tree, but it was there. We actually were the first to stamp in.
Hopefully they'll get that glitch fixed on the site so folks can find
the international boxes again. If you hadn't replied we never would have
known it was there. Thanks for the cute little frog stamp!
Nautilus
