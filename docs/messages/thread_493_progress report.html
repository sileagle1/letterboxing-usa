From: (Letterboxr@aol.com) | Date: 1999-05-25 17:22:24 UTC-04:00
Dear Group Members,
As promised, I removed the "moderate group" feature for this list. The only
address we continued to moderate was the one that had previously been sending
long argumentative messages to the list.

The next message from that address began, "This is an angry post!..."

The post was returned to the sender with an explanation that we would not
publish angry posts. Of the two moderators who were in place at the time the
message was sent, it was I who chose to return the questionable message to
its sender. It is my responsibility as the creator and manager of this
listserv to protect its 64 members from angry, destructive posts of this
nature.

This person has continued to send angry messages to many of our group members
personally. There is nothing I can do about this except recommend that you
contact him individually and ask him to stop. I'm very sorry for this
unfortunate situation.

I want you all to know that at this point in time, there are ABSOLUTELY NO
MODERATION parameters in place for this talk list. If the previous list
abuser or anyone else should again decide to start flooding this list with
angry, argumentative posts, then it will become necessary to apply moderation
parameters again. In the future, such decisions will be discussed on the
letterbox-vision talk list, and decisions will be made in a democratic
fashion.

Thank you all for your patience and understanding in this difficult time.
Mitch Klink, Portland OR
