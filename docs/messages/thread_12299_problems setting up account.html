From: Wes Garrison (wes@wesgarrison.us) | Date: 2003-04-25 00:13:29 UTC-05:00
A lot of people have been having problems with this. I would suggest that
you make sure you get the ENTIRE link. If you use Yahoo! or Hotmail, the
link might be cut off.

For instance:
http://letterboxing.wesgarrison.us/em.asp?a=a&e=wes@wesgarr
ison.us&blah=blah

Something like this would try to confirm 'wes@wesgarr' , which isn't the
address I want!

In this case, you'll want to cut 'n paste the first line, then add the
second line, then press enter in your browser to go to the page.

Hope this helps!
Wes
