[L-USA] The Letterboxing Web Ring is Growing!

From: Marty Chamberlain (martyc@cyberdude.com) | Date: 1999-06-02 16:00:03 UTC-04:00
A public thank you to the folks who have added their letterboxing web pages
to the newly formed Letterboxing Web Ring! There are actually 8 sites in
the ring already!

If anyone else on the talk list has a letterboxing site they'd consider
adding, please visit the webring (see address below) and submit your site.

I'd like to also remind those folks "across the pond" in the land of the
original letterbox that the webring is *NOT* just for North American web
sites. I'd love to have some UK (or other countries') sites represented!
Spread the word.

Thanks again,
Marty
