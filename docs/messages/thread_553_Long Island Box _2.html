Ed,
It's up to you. As long as your page is being mainly accessed from the LbNA
website, I wouldn't worry much about it. A reference to the waiver appears
on the main page (with the USA map), so if any litigatious soul should have
problems, we can always point out that they had adequate warning.

If you do want to reference a waiver, just add a link to
http://members.aol.com/catsdice/lbna/clues/waiver.htm

I don't recommend creating multiple copies of the waiver. As long as we all
link to the same page, we can decide as a group what needs to be stated
there, and we only have to update one page to make any neccessary changes.
If anyone feels the waiver at this page is inadequate, please bring your
concerns to this talk list. Then we can discuss its shortcomings and make
edits that will benefit the entire group.

Thanks for asking,
