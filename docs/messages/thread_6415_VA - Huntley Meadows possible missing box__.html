Hi!

My husband and I are new to letterboxing, and had chosen to find the ones
at Huntley Meadows last weekend as well. We, too, only found the second
one, but being so new, we figured we may have just missed the first one
(the Frog Box). The second one, the Dragonfly, refers to a "bat tree/J + B
heart" - never did find this tree, but did see a stump that we apparently
guessed correctly could have been the correct tree, because, according to
the directions ("look for a big tree bearing 300 degrees about 35 steps
forward") from this stump was the letterbox! And boy we were thrilled to
see it! :)

What a beautiful place Huntley Meadows is - we had no idea it was there,
right in the "middle" of civilization - a true treasure.

Take care!
