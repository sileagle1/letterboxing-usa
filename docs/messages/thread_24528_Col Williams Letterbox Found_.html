Cool! Glad it turned up!

I read some discussion somewhere recently, that the phrase "rehide
better than you found it" can be misleading to some and that it has
been misinterpreted to include "even if it's in a different

it means, but you never know!

something like, "replace box in the same location and recover better
than you found it".
