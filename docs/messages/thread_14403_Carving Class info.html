Subject: CT: Carving Class on 8/20

Back by Popular Demand!

The Stamp Studio in Manchester, CT is having a carve your own rubber stamps
class on August 20th from 6-8 p.m.

In this class, you will learn how to create and transfer images onto rubber
stamps. Unlike any other "carve your own" classes, we will cover techniques
like shadowing, texturing and shading. It's easy and fun for all levels. This
is a great class for those who have discovered letterboxing. (I did not add
that... it's in the brochure :-) Carving rubber will be provided, however, you
need to bring carving tools if you have them, or purchase them the night of
the class. The cost for the class is $22.

If you have any questions or if you would like to sign up for the class, call
Melinda at 860-432-1087.

Happy Carving!
Music Woman
