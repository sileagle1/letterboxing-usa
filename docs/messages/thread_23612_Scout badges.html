From: Michelle Lee (misch@pobox.com) | Date: 2006-07-28 08:45:16 UTC-04:00
Greetings,

The Connecticut Trails Council, Inc has a multi-level patch on
letterboxing (Brownies thru Cadettes). I have used it with some Girl
Scouts at my councils overnight camp and will be using it with my
troop next week. The only "badge" available thru GS USA is the "Hi-
Tech Hide & Seek". I have also helped a scout earn this badge. You
definitely need to be an older scout to earn that.

Yes, you can create a "council own" or "troops own" badge in
letterboxing. I looked into it, and I thought it was a lot of work.
I'm guessing that Girl Scout USA should be working on younger
levels soon enough.

I have ended up adopting a letterbox that a Girl Scout troop planted
and abandoned. They created it and then the troop disbanded, but
forgot to remove the letterbox :( A temporary box, is probably a
better idea.

If anyone would like the requirements for the Connecticut Trails
patch, please email me off this list.
