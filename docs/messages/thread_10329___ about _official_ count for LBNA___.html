From: Chuck Straub (woodschuckstraub@yahoo.com) | Date: 2003-01-17 18:11:03 UTC-08:00

Don't worry I won't argue with the forest frog family. I really like that name. We count Hitchhikers and event stamps as finds just as we would when finding a hidden box. If others disagree, thats ok but that is how I do my PFX count. It's my understanding that it was originally only PFX no E or HH. Psychomommy has all sort of counts that she does, Including lizards, and thats ok too. Thats her choice and I think we can respect that. There really isn't any set rules in this. I do understand how people would like a set of guidelines to go by but to my knowledge there are none. Well, anyway, thats what Molly and I do......Chuck and Molly
