Event Box Challenge: Reindeer Games

From: she_who_plants (she_who_plants@yahoo.com) | Date: 2005-11-04 07:42:29 UTC
Hey everyone!

Christmas is coming, and before it gets too busy I thought I'd bring
up an event box challenge--the first of many, I hope. I'm planning a
new traveling event box for those winter gathers that are hosted here
and there throughout the LBNA. It will be called "Reindeer Games" and
have eight stamps, each depicting one of the eight tiny reindeer
famous for pulling Santa's sleigh. Dasher, Dancer, Prancer, Blitzen,
Comet, Cupid, Donner, and Blitzen.

Here's how the challenge will work. If you have an idea of what a
stamp featuring one of these reindeer should look like, make a quick
sketch, scan it in, and email it to me. I'm going for things like
Dancer dancing, Cupid with wings and a bow...you get the idea. Keep
in mind that the finished stamp should be no bigger than 3" square.
I'll look over all the submissions and announce the winners...and
then those stamps will be carved and the event box will have been
created.

Does this sound like fun?
