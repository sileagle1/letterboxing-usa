From: Megan Thompson (myrtle888@hotmail.com) | Date: 2001-10-20 20:35:44 UTC-04:00
We found all 5 of these boxes today in good shape. Larry Boy and Dino Boy
loved this set because there were surprises in each box! Just right for
5-year-olds. The clues were excellent, the stamps are really nicely carved
and using the directions you could easily break this set up into 3 hunts of
about a mile or less. It sure was a beautiful day to be letterboxing!
Thanks to the Willington Gang for a great series of boxes.
