From: Wes Garrison (garrison@phys.ksu.edu) | Date: 2002-07-26 13:42:48 UTC-05:00

Some general group commands for those that don't want to log into Yahoo
Groups:

To put your email message delivery on hold for a group, send a blank
message to:
letterbox-usa-nomail@yahoogroups.com

To change your subscription to daily digest mode, send a blank message
to:
letterbox-usa-digest@yahoogroups.com

To change your subscription to individual emails, send a blank message
to:
letterbox-usa-normal@yahoogroups.com

To receive general help information, send a blank message to:
letterbox-usa-help@yahoogroups.com

Hope this helps.

Wes
