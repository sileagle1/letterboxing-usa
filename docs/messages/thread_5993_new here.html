From: arlen fletcher (arlenf@earthlink.net) | Date: 2002-07-04 21:59:23 UTC-07:00
Barbi,

Sounds like you're talking GPS to me! Try the Garmin Etrex Summit. It has
a built-in compass - not sure how accurate it is though, and it has an
altimeter too. Except for the compass part, it won't help much with
locating LetterBoxes. Really, a simpler compass with a movable bezel will
be quite adequate. I'm sure many on this list can give recommendations for
compasses which work well.

But here's a link to Garmin in case you're intrigued:

http://www.garmin.com/products/etrex/

Then click the product comparison link at the bottom of the page
