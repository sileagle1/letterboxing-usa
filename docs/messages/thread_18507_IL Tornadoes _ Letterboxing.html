I posted this on the Great Lakes board earlier today, but I figured
everyone on this board might like to know (if you haven't heard yet)
as well.

A number of tornadoes came through Illinois last night. 22 was the
last count I heard. Some towns were hit, mainly Utica. Downtown Utica
is a disastor right now, so if anyone plans on finding the Starved
Rock boxes, I don't know if you can get through the area or not.

Anywho...just plan on checking online newspapers of the city you plan
to visit to see if they had any damage. Some boxes might be in areas
that are off limits for the moment.

I'm from Rockford and we weren't hit, so the Red Rover box should be
alright, if you plan on visiting that one. Like I said, though, check
before you make your plans. :)
