Thanks Val for setting this up and for volunteering to "collect" for the 2nd
Batch of patches. :o)

I wanted to note, for the record, that the P & F denominations of 50
minimum are set by the supplier (who is not Town Hall Embroidery). Town Hall
is really the middle-man here. Town Hall is not a huge operation. They
have, probably, 8 computerized embroidery machines. which are mesmerizing to
watch. I literally have to move my children out of the store by their
shoulders. :o) Their bulk order supplier has the capacity to make patches
at a low price but to maintain this cost requires a 50 patch minimum of any
patch variation. Even something with simple variations like P to F. The
thing is that a 50 minimum for a $2 patch is the best I have found. It is
possible that a lower minimum exists but I haven't found it for the price.
:o)

T.H.E. has put a price together for me so that people can make individual
orders however the cost would be $10 per patch with an initial $80 computer
set up fee ... the good point was that interchanging the P to F to X is no
big deal so an additional computer set up fee is not required. Still, I just
couldn't justify the cost. Especially since these are the type of patches
that would be embroidered on sweat shirt material and then need to be cut
apart and/or trimmed. That is the type of embroidery work that T.H.E. is
set up to do. The quality would not be nearly as good. T.H.E. has been
very patient with me since the first order turned out to be a several month
project just getting the interest together and collecting the orders. I
waited for every last possible last minute order since I didn't know when we
would get enough interest together to make another one. :o) I was
pleasantly surprised to see the interest grow before the 1st order was even
sent out! :o)

I am still hoping for a merit badge order. I made a mistake, I think, on
the merit badge poll. I think I should have asked for a count per patch. My
suggestion would be to delete the first merit badge poll and submit another
one asking for the number per denomination. I know for myself I would order
atleast 7 per denomination. That would make a big difference if others had
plans to do the same. If no one objects I think I will do that and see how
it goes. If there is not enough interest then no harm is done. :o) I know
I felt bad having to return money to those ordering the merit badges in the
last order.

Sorry for another long post.
