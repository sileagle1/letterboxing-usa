Well, it was changed recently so the downloaded file is specified as
a CSV file instead of as an HTML file (which is what it was using as
default before!) Which, when you get right down to it, was flat out
incorrect.

There are settings on every computer that link certain file types to
certain programs, and for most people, opening a CSV file will cause
your operating system to suggest using Excel. Usually you can
override this, though I'm goofing around in England at the moment
and don't really have time to look up exactly the steps necessary to
accomplish this.

You could also open in the file in Excel and save it as a text file
easily enough as well.

In a nutshell, though, it's not the website, it's your computer. It
thinks CSV files should be loaded in Excel (which is *usually* what
most people want), but you want something else to happen. That
requires changing settings on your computer to change.

Happy trails! =)

Easiest way to work with the file: right click on the "Export Box
Info" link on the site, then select "Save Target As..." or "Save Link
As..." from the popup menu. Then save the file to your computer --
and change the extension to .TXT if you'd like. Then open the file
you saved to your computer using your text editor (Notepad).
