Do you mean you have a stamp you'd like to make into a box to send to events?

If you'd like to send an event box to an upcoming event, either post to the list to let folks know it is up for grabs or, better yet, contact the host of the event and arrange to send it to them. Be sure to have plans for what they need to do with it after their event...do they send it back to you or directly on to the next the event?

Good luck!

-Kim (Rustypuff)
