From: cpascott (seh-letterbox@comcast.net) | Date: 2005-10-05 14:12:04 UTC
Hi HER. Welcome.

In case you are not aware, there is a board called "newboxers" for
questions new 'boxers may have. This was set up by Mark Pepe to
reduce the number of repetitive questions on this board. Please feel
free to post any and all questions you have on that board --
regardless of whether they've already been asked. A number of more
"seasoned" letterboxers prowl that board and are happy to answer the
questions for you.

I'm glad to see this hobby spreading and the number of letterboxes
increasing across the nation. Of course, there are now plenty of
letterboxes that I'll never get, but I look forward to the day when I
can find a few outside of New England. Hmmm, based on my current "F"
count (435) and the number of letterboxes in the six New England
states (3,297 as listed on LbNA plus countless WOM boxes) it might be
awhile.

Anyway ... welcome again!


CPAScott
