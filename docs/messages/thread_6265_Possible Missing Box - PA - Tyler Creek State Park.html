Hello all! Discovered letterboxing from the in flight magazine on
US Air and have been loving it since! To all the webmasters and
box placers - thank you! Keep up the great work!

On a slightly sadder note, we struck out finding a box for the
first time this weekend and we think it might be missing! We were
hunting the Water's Edge box in the Tyler Creek State Park series
planted by Muk-Luk. We got to the spot where the letterbox really
looked like it should have been, but no luck! We hunted everywhere
near by that we thought it might be, but to no avail. If the
box's owner wants to contact me off board at
travelintiger@h...,
to see if we were just in the wrong spot, feel free. Or if someone
has actually found it recently, please correct me.

Keep on hunting!
