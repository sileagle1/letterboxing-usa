Re: CT-Lebanon Fair

From: mothermoo2001 (cstearns07@hotmail.com) | Date: 2003-12-18 14:24:04 UTC

> I was wondering if there would be enough interest in a series of
boxes at the Lebanon Fair in August.

We have done series at Tolland County 4-H fair twice with great
success.( 50 boxers stamped guest book) Last year we didn't do one so
we wouldn't conflict with a gathering in Mass or Maine can't remember
which now. Anyway we had planned to do one again this year but if
your doing yours maybe we won't our fair is usually the same weekend.
It worked really well and each box we did brought them to a different
part of the fair. Clues were in plastic pages in our tent and there
was only the one set for each box so folks wouldn't be hunting when
others had the box out. When finished they returned the clues to the
tent and picked up a new set. We had an event stamp there and guest
book as well and a few HH's passed through that weekend as well.
Have fun with it if you need any other ideas or help just contact
me. carol - Leader of the Pack
>
