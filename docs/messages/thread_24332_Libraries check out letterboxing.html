Our summer reading program (in Texas) is Set Sail for Reading with a
Nautical theme. There's a lot of variety (usually state-by-state) in
what they use for the program.

That said, I wish we were doing the Get a Clue program (sponsored by

would get into those themes and programs more than the Nautical ones.
