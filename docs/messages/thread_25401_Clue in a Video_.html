Clue in a Video?

From: (john@johnsblog.com) | Date: 2015-08-29 12:58:12 UTC-07:00

Did you know that if you create a letterbox clue as a video, it is very easy to add it as a clue on LbNA? You add the video to youtube, click Share, click Embed, copy the code give in the little text box, then paste the code into the clue area on letterboxing.org when adding a letterbox. Done!
