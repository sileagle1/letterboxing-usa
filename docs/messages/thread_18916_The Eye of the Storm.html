I'd like to thank everyone who's written to me concerning the recent thread about the 'All Things Considered' titled
commentaries I've written and the legal ramifications of using that phrase.

Given the fact that I really don't want to start another flame war and cause people to raise up in arms over this, I
have decided to NOT continue pursuing NPR for written consent to use the service marked (Yes, I did do my homework
prior to choosing that name) 'All things considered' name.

I appreciated Wes's point about using an original name, and, since there seems to be a group that tries to tear down
anything that I build or do, I've decided to change the name of the commentary that I write.

Look for the latest installment of "The Eye of the Storm" this Wednesday at it it's usual location, along with new
commentaries from other writers as well.
