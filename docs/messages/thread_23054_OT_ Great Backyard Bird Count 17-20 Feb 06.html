This is a bit off topic but it may interest letterboxers out there
because it involves birdwatching, spending time outdoors and it's a
great way to introduce folks to birdwatching.

The annual Great Backyard Bird Count is scheduled for 17-20 Feb 06.
Check out: http://www.birdsource.org/gbbc/ for more details. This may
work
well with letterboxing adventures, too--since many of you will already
be out and about.

If anything the website provides a link to Cornell University's
incredible online bird guide. Great resource for letterboxing, too.

http://www.birds.cornell.edu/programs/AllAboutBirds/BirdGuide/

Enjoy!
