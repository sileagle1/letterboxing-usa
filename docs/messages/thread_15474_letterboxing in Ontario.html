If the date that the box was planted is unknown, it shows up as 1969 in the
database. Besides contacting the owner (if possible), the only other thing
you can do is search the messages of this list on the Yahoo groups site,
and see if there have been any reports about that particular box. Searching
is slow, and Yahoo only searches the newest messages first, so you need to
keep clicking "Previous" (or is it "Next"?) to run the search on
progressively older messages.

Otherwise, hope to have a nice hike and maybe you'll get lucky and find a
letterbox along the way!
