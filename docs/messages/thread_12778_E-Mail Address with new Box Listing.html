E-Mail Address with new Box Listing

From: SpringChick (springchick@letterbox-mi.com) | Date: 2003-05-11 12:50:11 UTC
Wondering how others feel about asking Wes if a field could be added
to the new box records for contact e-mail address?

One of the things I have found with the new system that if a box is
placed by a user name that I don't recognize and they don't give an e-
mail address in the clues, there is no way to get in touch with that
person with questions about the box or if you find the box and it is
in need of attention.

With the old system, there was always a clue post to go back to,
although it is true that if significant time had passed, the e-mail
may have been obsolete.

I think most people just don't think about putting their e-mail
address in the body of their clues. If there was a field on the
entry form, it would prompt them for this information, and I think
most would be okay with providing it. It could be an optional field
if some do not wish to give out their e-mail address.

Just curious if how other people felt about this?
