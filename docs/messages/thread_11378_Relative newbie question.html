How you count your finds (& if you count them) is personal preference, but
if I have to follow separate clues to a box, I count each as a find. It
would be like doing a series of boxes that are all in different locations
(you just happen to get one image from them). However, if a box contains
multiple stamps, I count it as one. Except in the case of a hitch hiker. :-)

No rules... except to have fun!
