CT: Walk through Geologic Time LBs

From: Magic Mike (mmletterbox@hotmail.com) | Date: 2002-09-13 22:13:59 UTC
Had an opportunity to go find the Walk Through Geologic Time Letterboxes at
Dinosaur State Park in Rocky Hill, CT.

It was a great day and a wonderful nature walk...

The Trilobite Letterbox was a bit damp and the book is filled, but I was
able to squeeze my stamp in a place.

The Eubrontes Letterbox is in bad shape...the logbook was still there, but
someone swiped the stamp and pad. Too bad, the picture of the stamp in the
book shows that it was a good stamp.


There is also the Native Flower Letterbox there, but it is a different
person placing it and a different set of clues...That box is in great shape.
