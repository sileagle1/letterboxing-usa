>>AND they sell the foam adhesive for stamp mounting.

We found that when using the pink stuff, they easily mount with a dab
of super glue to the top of the ink pad. We use the full size ink-it-
up (or something like that) pads with the detachable tops (some
white, some clear) and it eliminates inked up hands. Before mounting,
make sure the whole assembly will fit in what ever sized box you
choose -- some of the smaller containers are not deep enough to do
this with.
