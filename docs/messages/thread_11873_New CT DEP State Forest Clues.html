New CT DEP State Forest Clues

From: (cmdonnelly@aol.com) | Date: 2003-04-11 18:57:10 UTC-04:00
The foresters on Connecticut's State Forests have been busy placing
additional letterboxes on the State Forests, as part of the celebration of
our Forest Centennial. In case you were wondering, each letterbox is placed
by the state forester responsible for that forest (each forester has the
responsibility for several state forests). Not only do each forester choose
to the site, he or she writes the clues. This has become a very fun and
intriguing project to them!

The six new forests are:

People's State Forest, Barkhamstead, CT (Litchfield County)
http://www.dep.state.ct.us/burnatr/forestry/centen/peoples.htm
placed late March

Nehantic State Forest, East Lyme, CT (New London County)
http://www.dep.state.ct.us/burnatr/forestry/centen/nehantic.htm
placed late March

American Legion State Forest, Barkhamstead, CT (Litchfield County)
http://www.dep.state.ct.us/burnatr/forestry/centen/amlegion.htm
placed late March

Shenipsit State Forest, Somers, CT (Tollan County)
http://www.dep.state.ct.us/burnatr/forestry/centen/shenipsit.htm
placed late March

Salmon River State Forest, Marlborough, CT (Hartford County)
http://www.dep.state.ct.us/burnatr/forestry/centen/salmonriver.htm
placed late March

Nathan Hale State Forest, Coventry, CT (Tolland County)
http://www.dep.state.ct.us/burnatr/forestry/centen/nathanhale.htm
placed late March
