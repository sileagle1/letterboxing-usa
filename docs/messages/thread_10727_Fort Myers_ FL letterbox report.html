Fort Myers, FL letterbox report

From: Doug Gerlach (silentdoug@douglasgerlach.com) | Date: 2003-02-09 18:30:49 UTC-05:00
Today I found the Caloosahatchee Paddler, Mud Cutter and Land Shark
letterboxes in and around Fort Myers, FL. All were a bit damp inside, so I
rebagged when possible and left dessicant packets with the logs. Land Shark
was really the worst; the journal was quite waterlogged and should probably
be replaced.

There was only one other box in the area that was potentially accessible,
but it could only be reached by water and I was pressed for time on this
trip. Maybe I'll tackle that one next time....
