Everyone seems to have a different way of doing this. Team Rayvenhaus does
it this way:

1) We count those as [A]dopted, which is where the A comes from in our
count.
2) We count those as [E]vent stamps and that's where the E comes from in our
count
3) We count those as [T]ravellers and that's where the T comes from in our
count.
4) Most Travelers are that we are familiar by, though this in no way means
that all Travelers are. We've had to meet someone and give them a code
phrase, we've merely had to be at an event and we've had to find someone at
an event and give them a code word. It all depends.

Steve
