From: ontario_cacher (ontario_cacher@yahoo.ca) | Date: 2006-01-07 22:14:30 UTC
I'm looking for virtual letterboxes in the LBNA database. I tried the
little computer icon on the Map page which is supposed to go to
http://www.letterboxing.org/virtual/virtual.htm but I get a "404 not
found" message.

I'm interested in posting and finding virtuals but I'm not sure how to
go about it. I posted one virtual but had to choose a county and
nearby city, there was no "virtual" option. Any advice?

Re: [LbNA] Re: Map - virtual letterboxes - link doesn't work

From: JARS (ontario_cacher@yahoo.ca) | Date: 2006-01-09 09:52:14 UTC-05:00
Thanks Don. I tried that last night and found a few. I also figured out how to post virtuals by choosing county: mystery and naming the nearby city: virtual.

Virtual boxes are difficult to find. You have to know to do a keyword search and the word virtual needs to be in the title (or clue - but that can be messy, it picks up non-virtuals that use the word 'virtual' or 'virtually' in a different context).

Do you think that LBNA might someday provide a 'virtual' option similar to a 'mystery' option. I sure hope they get that little computer (virtual) icon working on the map page. It would be nice to have a list of virtuals (i.e. separate the virtuals from the mystery boxes).

Right now I post my virtuals on Atlas Quest which has a separate option for virtuals. It's a great site but some Mac users have trouble with AQ so I thought I'd post them on LBNA also. Posting them on 2 popular databases will give them more exposure.

JARS
