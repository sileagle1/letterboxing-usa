No. If it makes it to the talk list, it will eventually make it to
the LBNA Web site because notices of new boxes are culled from the
talk list.

However, there are an undocumented number of cuckoo or bonus boxes
throughout the country. These are spread by word of mouth or a boxer
finds a box that IS listed on the LBNA Web site that also contains
the location of a cuckoo box.
