endusers,

importing bulk records is fairly easy using Excel, however, MS Access
is fickle when it comes to matching column for column. if you are
importing data for the first time the best way to make sure your
columns line up it to "Export to Excel" an empty spreadsheet. the
export feature will automatically carry over the column headers in
Excel. import your spreadsheet using the "Log Book Datasheet" option
in the main menu.
