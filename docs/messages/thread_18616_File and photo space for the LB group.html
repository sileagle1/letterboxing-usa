Attn: Letterbox-USA members.

That time has come again when we are coming close to running
out of storage space in our group site. Some of the photos and
files that are in there are repetitive or not very applicable to the
group as a whole. The files section, in particular, is quite full.

We are asking for you help in reducing the storage space
problem. Could you review additions you have placed in there
and see if you feel we still need them. If you do this for us, it
saves us from having to delete files that others may wish to have
there.

Remember that an easy way to solve this problem is to post the
photos or files on an independent website and then you can post
the link to the list. It uses much less memory that way, and it is
easier to do than some might think. There are many websites
(such as Yahoo/Geocities) that will take you step by step through
the process and you don't need to know anything about
programming codes.

We would really appreciate your help with this concern.

-Cadenza
