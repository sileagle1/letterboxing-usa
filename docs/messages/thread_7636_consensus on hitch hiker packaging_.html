From: dvn2rckr (dvn2rckr@yahoo.com) | Date: 2002-09-12 03:44:13 UTC
I don't think it matters. Remember, the smaller it is, the easier
it is to hide at various locations (especially if you have to co-
locate two large containers). Some folks think that if the HH is
only in a ziploc bag that the stamp might be more likely to become
separated from it's log book. I've found both types in my
experiences and don't really prefer one over the other. Both
methods seem to work quite nicely. If you go the ziploc bag
approach, make sure that EVERY COMPONENT is well marked that it is
part of a hitchhiker set!!! You might want to include an info sheet
specifically written for hitchhikers as some folks may not know what
they have when they find it!
