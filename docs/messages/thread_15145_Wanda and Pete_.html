From: Bigguy (bigguy9210@yahoo.com) | Date: 2003-08-16 13:49:09 UTC
We count our own hitchhikers in our P count when we release them but
not existing hitchhikers when we re-release them. We count those in
our F count. We didn't figure it was right to count the same stamp
for a P and an F at the same time.

Sorry - I forgot to answer the first part of your question. Yes,
