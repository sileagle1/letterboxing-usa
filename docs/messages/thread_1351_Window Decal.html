From: (bsennott@crocker.com) | Date: 2000-10-12 15:17:40 UTC
Putting "org" in a different color is a nice idea, but as an editor,
I
have to weigh in and say, please don't capitalize anything in the
URL.
A string of capital letters (like URL) indicates an acronym, which
the
"org" in www.letterboxing.org is not.

URLs are normally typed like this: www.letterboxing.org. It can be
either roman or italic.

Bonnie
