CA: The Return of the Eaton Hummingbird in SoCal

From: trentlaudes (trentlaudes@excite.com) | Date: 2004-02-10 05:18:57 UTC
I'd like to announce the return of the Eaton Canyon Hummingbird
Letterbox. This LB was orignally placed on 1/1/03 by twofamiles, but
was lost several months later. This was my 2nd found letterbox, and
the 1st one I ever stamped into (I found the 1st one, "The Eaton
Canyon Letterbox," a week earlier before I had a stamp). I've
recarved the stamp in the image of the 1st one (a bit bigger), and
placed it in the orignal spot, using the clues of the orignal
placers, so I don't consider this to be a new letterbox find or
placement. The hike to these falls was taken and written about by
John Muir in the 19th century, so you'll be literally travelling in
his footsteps.
