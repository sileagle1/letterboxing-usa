From: abacus00 (abacus00@yahoo.com) | Date: 2003-08-10 13:34:16 UTC
Especially for those who read the list as email, you need to
visit the group website in order to access the database.

http://groups.yahoo.com/group/letterbox-usa/

Then you will have a menu bar on the left: Home, Messages,
Post, Chat ... on down to Database. Follow this link and then
choose the Box Status Database.

You can also find an extract of the database as an Excel file
in the Files section.
