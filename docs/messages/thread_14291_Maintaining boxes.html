No rules. Generally I check on my boxes if I'm in the area or if I hear
there is a problem. Probably every 3-6 months otherwise. I always enjoy
getting emails from people visiting my boxes - and it helps if they say how
the box is getting on.

> Has anyone tried putting a note to the owner in the box about
> the abandoned
> box situation that's going on as well as emailing the
> individual?

Yes, there are a huge number of emails circulating the internet as we try
and track down owners. A lot of folk were having problems registering with
the site and so now that we know we can help them out. I'm encouraging
potential adopters to go visit the box and see if there are contact details
in the log book. I would greatly encourage anyone who ends up adopting a box
to leave a note in the box for the box owner too.

> Does anyone
> know how long the webmasters are giving before the clues
> disappear from the
> site?

No set date as far as I know. We're doing our best to make sure no box
remains abandoned. Everyone can help with this by tracking down owners in
their area (a new form of letterboxing perhaps where we find the boxer
rather than the box??).

> Also can someone tell me the difference between the > HREF="www.letterboxing.org">www.letterboxing.org site
> and the letterboxing.wesgarrison.us site.

The latter was the temporary site address for the new website while it was
under construction. It has now been transferred over to the original site
(letterboxing.org) and the old one is now under http://www.rpmrlm.com:1020/.
So www.letterboxing.org is the one to use.

Hope that helps some,
