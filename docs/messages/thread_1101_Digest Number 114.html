I know how much a pain alot of email can be, however there are a few
options you may want to consider. Firstly, many email programs include
capablities to filter email into a folder. This is what I did so that I
read LbNA materials when I feel like it. (which is usually still
everyday.) Just filter the subject heading for text containing
"[LbNA]". I use unix/pine, so this is sort of trivial. However I am
aware that outlook, hotmail, eudora etc. also contain the same
capabilities.

Secondly, I don't know if you have noticed this, but there is a footer
link at the bottom of every LbNA email which contains information on how
to unsubscribe.

The short answer: send blank an email to:
letterbox-usa-unsubscribe@egroups.com

I know that on letterboxing.org there is instructions on how to do this.
It is located in the Join link, in fact it is directly under the subscribe
form.
