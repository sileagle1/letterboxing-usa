If you cut/paste this:

C:\Program Files\Common Files\Microsoft Shared\PhotoEd

into a Windows Explorer Address, you should see it. Right click it and
send to the Desktop as a Shortcut. I've never used Picture IT. Does
it have the STAMP function? If you don't have a it, you can increase
the Contrast to get the same result.
