Fellow letterboxers,

At the request of our friend Kay (aka "Mohmers"), I have created a set of
printable online forms for ordering LbNA logo patches, window decals, and
merit badges.

Please note that the logo patches and window decals can only be ordered IN
BULK. As always, group orders may be arranged on this talk list whenever
someone is willing to organize a bulk order.

It is my understanding that individual orders are welcome for the merit
badges. Just print out the order form, fill in the circles with the letter
designations and denominations you want, and send it in with your payment.

All of the order forms can be found at:
http://www.letterboxing.org/regalia/orderforms.htm

I will likely add this link to our main website permanently, whenever I get
some free time.

Enjoy!
