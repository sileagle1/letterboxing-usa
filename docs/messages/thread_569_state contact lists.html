This is a reminder to people who may be state contacts
(subscribed to one or more of the letterbox-xx contact
lists linked from the map).

When a new person sends e-mail to the contact list, you
have to reply to the person directly, not to the contact
list, because the new person won't be on the contact list
and thus won't get the reply. The problem is caused by
the fact that the default reply-to on those contact lists
(at least the ones I'm on) is set up to go to that list,
when it should be set up to the sender -- so you will have
to type their name manually (and cc: the contact list if you
want, so other contacts for that list know what is going on).
