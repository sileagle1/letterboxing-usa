There's a new virus out.

It is supposedly from Microsoft, alerting you to update your programs by running
the included attachment.

Possible From: names are:
MS Security Services
Microsoft Program Security Division
Microsoft Corporation Security Section

Subjects for this email could be:
Latest Internet Security Upgrade
Latest Net Critical Patch
Microsoft Security Update

In some cases, the email looks very authentic, because they've used the format
of the Microsoft website.
It may even come from an email address that ends in @microsoft.com.

Remember, never run programs that are attached to your email.
Even if it's from your best friend, wife, husband, child, cousin, or dog, that
doesn't necesarily mean that
they actually sent it to you.
