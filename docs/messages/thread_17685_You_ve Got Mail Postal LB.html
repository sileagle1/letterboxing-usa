Snail Lady,

Retyping your address doesn't help. Anytime someone types anything
that looks like an email address, Yahoo disguises it by chopping it
off. This is to protect you from having your email snatched by
automated programs that search the internet looking for addresses
for spam lists.

You need to type something like this:

"My address is szorzi_1999 (and then you readers should type the
symbol for "at") yahoo (and then type a period) com."

Since we were able to see Rturtle99@a, you can for now just let us
know what comes after the "at" symbol and we'll be able to put it
all together!

Thanks,
