Clear your temporary internet files and it will work again.

I think there is something wrong with the way the website is updated
that doesn't change the date and time stamp on the files properly.
So when your computer checks the time stamp with cached files it
improperly chooses the cache, which no longer works. Clear the
temporary internet files and it should work again.
