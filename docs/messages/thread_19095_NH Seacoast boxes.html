Phyto and I searched for these boxes last May. We had
no luck in finding "Hear the Sea" either. I recall seeing
several other posts on the list saying the same.

We did find "Portsmouth Harbor Light", which I recall
being a fairly challenging find, as the bearings in
the clue don't quite work. See message 34371 for a more
recent find of this box (last October). The box was
fairly well hidden. I would not be suprised
if it is still there, lurking in the rocks.

Happy Hunting,
