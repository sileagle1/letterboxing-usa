Award-winning board game about letterboxing available

From: Ron Hale-Evans (rwhe@ludism.org) | Date: 2006-08-22 10:34:19 UTC-07:00
Hi all,

I'm a Seattle-area letterboxer who recently created a board game about
letterboxing called Piecepack Letterbox. The game won first place in a
game design contest this month, beating out 16 other entries. The
rules are here:

http://www.ludism.org/piecepack/pplb.html

To play Piecepack Letterbox, you'll need a set of components called a
piecepack, plus the nature-themed 4 Seasons Expansion. You can make
these yourself or buy them online. You can learn about the piecepack
and expansions here:

http://www.piecepack.org/

I plan to release a commemorative box soon called (what else?) the
Piecepack Letterbox.

Please let me know how you like the game and how well you think it
captures the feel of letterboxing.

Ron H-E
