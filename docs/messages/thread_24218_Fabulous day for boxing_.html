Fabulous day for boxing!

From: rozebud.rm (rozebud@rocketmail.com) | Date: 2007-04-08 03:09:26 UTC
We spent the afternoon hunting and finding The B Hunters spectacular 11-
box series "The Rains Came Tumbling Down". Seriously, folks - if you
find yourself in the STL area with a day to kill, go get this series!
Intricate carvings to hunt via the epic poem clues! Thanks so much, B
Hunters!

A bonus - when I got on line tonight to log these finds and write a
thank you note, I had a note in my AQ mailbox - seems a box of mine
which had been reported missing - AND I verified it's missing status
Thursday - - - was found on Friday! Whaaa...? How could this happen?
I'm pretty darn sure I was in the correct spot when I checked on it -
there aren't a lot of hollow pines in this particular location. Not
that I'm complaining, mind you...I was sad to see this one go, as it
was a favorite carve, and getting it back, unexpectedly, is a real
treat. But still. Whaaa...? *scratches cranium* Lucy! You got
some 'splainin' to do!

~rozebud
