From: Bigguy (bigguy9210@yahoo.com) | Date: 2003-08-26 22:53:52 UTC
I can't answer for all the missing database entries, but I can tell
you that I removed my 50-some entries yesterday. When it was
proposed, I said I didn't think it would work - although not for the
reasons that transpired. I just thought not many people would bother
to post their finds - it does take a bit of time. Nevertheless, I
did my personal best to help it succeed by posting all the finds we
had made so far. I never envisioned the extreme opposition from some
box-owners. Some of them have some very thoughtful, well reasoned
logic for their opposition - some are a bit over the top. In any
case it became clear to me that it can not work in it's present
form. The poll results made it clear that there is a significant
level of opposition and the steady stream of 'Don't list my boxes'
messages as well as vandalism of the database demonstrated that this
database can never be as comprehensive and accurate as those who wish
for a database want it to be. Therefore, I have removed all my
entries from the database. I had already added status lines to my
clues - for now I think that is the best we can hope for.
Bigguy
