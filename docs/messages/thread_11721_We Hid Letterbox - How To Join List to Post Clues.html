From: SpringChick (letterbox@attbi.com) | Date: 2003-04-06 15:59:09 UTC
Clues should be posted to the talk list with the entire clue in the
body of the e-mail, as the list does not accept attachments.

The subject line should indicate the state and that this is a new box.

Begin the message with the basic info about the box:\

Box Name
City, State, County
Date planted and by Whom
Difficuty Rating (optional)

As an alternative, if you have your own website, you can just include
the basic box information and a URL link to the page that contains
the clues.

Deb (SpringChick)
