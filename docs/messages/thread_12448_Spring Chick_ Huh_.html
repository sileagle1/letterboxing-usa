From: SpringChick (letterbox@attbi.com) | Date: 2003-04-29 20:25:26 UTC
Okay, I'm not exactly sure we're on the same wave length here...

Along with the new site, there is a new Yahoo group called lbox-
announce. When you add a new box at the new site, you have the
option of "announcing" the box to this list. If you check the box,
it will automatically send a message to the lbox-announce list about
your new box.

As with any other Yahoo group list, once you are a member, you have
the option of receiving the messages in individual e-mail messages, e-
mail digest form or to not receive e-mails (in which case you would
probably view them online).

By setting yourself up to receive individual e-mail messages from
this group, you can then route the messages to specific folders once
they are received by your e-mail program using the rules feature
(this works with Outlook and Outlook Express but I don't know about
others).

Are we talking the same thing? If so, sure I would be happy to try
and outline the steps to do this.

Deb (SpringChick)
