I don't think its up to the webmasters, but up to the list (or the
letterboxer that want these things). The webmasters aren't interested
in setting policy for the hobby.

I think the question is how many different denominations can be gotten
in one run? I don't know how these things work, but if the limit
is 5 denominations per run, I would throw the 75 and/or the 10 overboard
and replace it with a 500 and 1000. The 75 definitely seems useless,
IMHO :-)

It also depends on how frequently one of these orders can be mustered.
If it is more than once a year or more, getting 500s at least in the
pipeline now makes sense.

So, if it were me, I'd do 25, 50, 100, 500, 1000, and sort of make
that a standard, for now. But like I said, I really don't understand
how these things work. If we can do 10, 25, 50, (75), 100, 500, 1000
now, fine, but I personally would order a 500 if one were possible.
