Old Hickory Letterbox

Jackson, LA

East Feliciana Parish

Planted 8/11/2002 by Jim. Dillon and Family

The Old Hickory Box should be easy to locate. You can practically drive up to it.

You must locate the Old Hickory Railroad depot. There is a road that runs between the tracks and the museum. On the museum side, you will notice some trees where the road turns to the west. You will find the box beneath the tree closest to the road.

For more information about the Old Hickory Railroad, follow this link:

http://www.felicianatourism.org/train.htm
