Mount Bieterman returns!

From: Robert Hamel (hamel@ll.mit.edu) | Date: 2001-12-27 00:36:57 UTC-06:00
I am pleased to report that the Mount Bieterman Letter Box in Glendale,
Wisconsin has returned to active duty. It had apparently been attacked
by an indigenous letter boxer. The residue was gathered up by Bob, a
local two legged letterboxer (many thanks). Yesterday it was repaired
and replaced in the same location in a make-shift rodent-proof
container. It should be safe from all rodents save those with tool-steel
teeth.

Birkie Bob
