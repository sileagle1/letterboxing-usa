[L-USA] Letterboxing Web Ring

From: Marty Chamberlain (martyc@cyberdude.com) | Date: 1999-05-26 15:27:18 UTC-04:00
Hi group! I thought the web ring was such a great idea that I've gone
ahead and set one up. The idea is that if you have a web site that
features letterboxing (in most any sense), we can all link the sites
together so that people can browse through them easily.

If you have a letterboxing-related web site, and you'd like to be in the
ring, please e-mail me the URL of the site. Once your site is added to the
ring, you'll just have to put a little piece of HTML code on your page to
facilitate browsing around the ring.

After I've got the initial group of site on the ring, I'll set up an
automatic form on the ring homepage to allow easier submission of web sites.

Thanks,
Marty
