Hello,
I have the cost and info for the LBNA patches and window decals.

The window decals are $4 each on orders of 25 or more...2 colors...they will
look just like the LBNA logo and wording found at the top of the LBNA web
page.

A 2"x2" LBNA logo patch is $2.00 for 50 or more.

the P# & F# patches are 1-1/2" round badge patches also $2.00 for 50 or
more...divisions are P10, P25, P50, P100, P150 same for F#. These patches
have the letter & number and the North arrow for the Logo (the "A" in
America) in the center.

If you are interested in buying these patches send me a note with the
quantity.

This is the problem.....with other patches being offered there may not be
the same interest in these LBNA patches. I had a list of more than 80
patches from the summer. I need to be contacted in the next few days by
those who want these patches.

Sorry that I'm going staight to the list and bypassing the web masters on
this...I thought this would be quicker.

Please respond as soon as possible. YEa or Ney.

Next... if quantities of 50 or more are not possible the next option is a
slightly more expensive patch on cloth which is a little less stiff.

Please respond.

Thanks
