How to get help with the new site

From: Wes Garrison (wes@wesgarrison.us) | Date: 2003-04-26 01:49:08 UTC-05:00
Please, help me out when you ask questions.
These things will help me help you.

- Read the FAQ before asking a question. It's being updated daily with new
questions.

- If you're having a problem with a specific box, let me know the name of
the box you were working on, along with at least a state.

- The 'move-a-box' feature is only available for boxes that have already
been entered by the webmasters on the new site.
If your box exists on the new site, I need your member name in order to
transfer your boxes to you! Also, a list of box names helps. If you say,
"Give me my boxes." I'll probably just write back and ask for a list.
If your box is only on the old site, we'd prefer that you create an account
and put it in yourself.

- Email me off list, please. It's just a lot easier for me to keep track
of things when I don't have to wade through the main list. Plus, we don't
need to clutter the main list with small issues like "Give me my boxes."


Other notes:
- Thanks for all the great suggestions!
- If you see counties that aren't filled in, please let me know about those
as well.
- As of this email, 1360 total boxes have been entered. Cool. Northeast
Connecticut leads the way with 158.
