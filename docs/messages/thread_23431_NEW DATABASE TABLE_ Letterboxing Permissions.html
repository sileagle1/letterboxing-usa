NEW DATABASE TABLE: Letterboxing Permissions

From: StarSaels (steves_1701@yahoo.com) | Date: 2006-06-03 02:32:56 UTC
Dear North American Letterboxers,

This message is to announce that a new database table has been created
to allow the users of this group to compile a list of places, parks,
regions, counties, states, etc. where letterboxing is or has taken place.

I have taken the liberty of adding in a few select places so you can
see how the fields should be filled in. If you have important
*positive* comments, please use the proper field... for example, CT
parks who participate in letterboxing should receive a comment in the
"Authorities Aware?" field.

Please show your support for the letterboxing community as a whole by
adding your personal knowledge to the file. The idea behind this
"list" is to give ourselves reasonable, statistical ammunition to
fight the good fight for the freedom to go letterboxing!

"Letterboxes - our parks wouldn't be the same without them!"
* StarSaels

With this we must all remember that Letterboxing is not necessarily a right but more of a priviledge. Priviledges can be taken away if they are abused. We must remember the true spirit of Letterboxing where "i've found a neat place and i'd like to show it to you!" However we should not go trouncing through the woods, public, etc. assuming that we can put our plastic boxes just anywhere.

~The Ram~
