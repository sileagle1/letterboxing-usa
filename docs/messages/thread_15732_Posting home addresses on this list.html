You can delete your own posts and it is probably a good
idea if you have posted sensitive data that you would
rather not have available forever. First make sure you
are signed in, then open the message you want to delete.
At the top of the page is a band that starts "Welcome",
under that is "letterbox-usa" and under that is
"Messages". The next line starts with a link for
"Reply". When it is your own message, the line ends with
"Delete". Push this link to delete the message.
