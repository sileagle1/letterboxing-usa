From: Silent Doug (silentdoug@letterboxing.info) | Date: 2005-06-02 17:08:29 UTC-04:00
At 03:06 PM 6/2/2005, you wrote:
>I keep getting a "Connection Refused" error at both the LbNA and AQ
>websites. Anyone else having trouble?

It's been discussed before -- both sites coincidentally use the same web
hosting provider, so if a problem affects the web host, then both sites are
likely to be affected (or, if you look at it another way, if both sites are
down at the same time, it's not the web sites, but likely the hosting
company's servers). These kinds of problems tend to get resolved in short
order, so the best thing is to be patient and try again later.
