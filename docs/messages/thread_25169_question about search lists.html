From: Debbie Kotlarek (kotlarek@wi.rr.com) | Date: 2010-01-14 23:16:24 UTC-06:00
I assume you are talking about the box listings on letterboxing.org. If you
order the list by Location, I believe the boxes are listed alphabetically
first by county, then by city. You can choose to sort by Placement Date
instead, then they will be listed with the most recently planted box first.



Another option you might want to consider is exporting the list to a
spreadsheet. If you do a "Search for Boxes" based on some criteria, you
should see a "Export Box Info -
export to Excel in .csv" option near the bottom of the page.



One of the webmasters might be able to provide more info on this.
