Hi - just a reminder that if you plant a box on Sunday, September 19th -
which is International Talk Like a Pirate Day - it would be very cool to
write your clues in Pirate Talk. An example may be found in the clues to
Leapin' Lovers, western NC.....planted Sept. 19, 2003.

There are Talk Like a Pirate Day celebrations all over the world this
weekend - including Baghdad! For more information on that, and for
instructions on how to Talk Like a Pirate, see www.talklikeapirate.com
