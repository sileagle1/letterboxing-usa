Hullo, all! For those considering leaving the BIG chat list because it's gotten less letterboxy, please notice that there are a number of letterboxing chat lists specific to geographical regions.

I've only been on Letterboxing Northeast for a comparatively short while, but so far the topics have been germaine to the hobby: updates of box and trail conditions, notice of events, and hints for better letterboxing.

I suspect that regional chat lists may be more polite because we've met each other, and put a face to many names.

I offer this idea not to torpedo the USA chat, but rather because I see a number of people *who are uninvolved in the flame exchanges* saying "Wow - this place is rough. I'm going away!".

Just a thought. Open fire.
