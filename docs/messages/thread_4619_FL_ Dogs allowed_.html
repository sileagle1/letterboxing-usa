I am not sure about the Canaveral marshes rules, but I think it would
be OK. There is a board at the entrance with rules posted as I
recall. It is a _very_ primitive park, no facilities of any kind. I
have only seen a ranger there on one occasion. I certainly would keep
her on a leash though, too much to get into out there. If you go to
Blackpoint while you are in the area it is a wildlife drive so that
shouldn't be an issue. There is a hiking trail there too, but I don't
recall what the rules are. The Blackpoint letterbox is on the drive,
not the hiking trail.
