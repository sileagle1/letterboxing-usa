You can. Click on the little green flag and check it
out. Also, AtlasQuest has that feature as well...

It isn't really a problem with someone giving the
location away because there isn't room to comment.
Hopefully, a finder will use the "contact the placer"
feature and email the placer directly with any
comments.

The only downfall to the feature is that if folks
don't use it when they find the box, you still won't
know it's been found. Based on the notifications I've
gotten and the number of stamps in my book, I would
say that MAYBE 25% of the finders either email me or
update the clue page to say they've found it.
