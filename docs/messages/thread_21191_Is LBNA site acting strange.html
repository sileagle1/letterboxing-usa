I have noticed that boxes don't show up when you navigate through the map
unless you entered the county name exactly the same as LbNA expects the
name. This is a common problem in Lousisana where most county include
"Parish" but LbNA doesn't include it, e.g. "Jefferson Parish" is listed as
"Jefferson" county. These boxes will show up when you search for all boxes
in the state using the Search for Boxes link.
