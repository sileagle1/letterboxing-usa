From: catbead1 (libby@twcny.rr.com) | Date: 2004-02-09 21:23:17 UTC
There's an easy way to reduce the size of images in Photo Shop (5.5)
that won't diminish the viewing quality. I use it all the time for
computer images & jewelry images. On the top menu bar, go to the
"File" > "Export" > "GIF89a Export..." follow this, it really reduces
image size but keeps the clarity.

Hope this helps,
