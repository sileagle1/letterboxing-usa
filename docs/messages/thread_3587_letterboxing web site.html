From: Tom Cooch (tcooch@sover.net) | Date: 2001-10-29 18:19:04 UTC-05:00
Scott,

At this time, there are four of us who have responsibility for posting new
boxes to the state maps. I think that is happening relatively quickly. Jay
Drew maintains the What's New page, and is also very quick there; he had
some computer problems for a few weeks, but I think those are now resolved.

If you think that there has been a new box posted to this list that has not
been put up on either the What's New page or the appropriate state map after
a few weeks have gone by, please just post a reminder to us all.

Tom
