From: Debbie Kotlarek (kotlarek@wi.rr.com) | Date: 2005-06-16 20:57:39 UTC-05:00
I agree - I love all the new features!

Could I ask for one more? When we're planning a trip, I copy & paste the
box listings for a region into an Excel spreadsheet. This gives me columns
for Box Name, # of Boxes, Nearby City and County. We can then organize based
on our route. I usually add two more columns to my spreadsheet, one for us
to estimate the time or mileage for a box and the other for us to write in
comments about the box so we can give a status report to the placer. This
is where my request comes in - could you make a listing that also includes
the placer? I know if I "Search for Boxes" the Placer name appears (but not
the county). I'd really like to have all 5 fields appear, because that
would then allow me to sort my spreadsheet by Placer name when we get back
home. I could then more easily report on all boxes we looked for that were
placed by each person.

Thanks for considering this request!

Wisconsin Hiker

From: (john@johnsblog.com) | Date: 2005-06-17 08:31:55 UTC-04:00
Wisconsin Hiker,

I recently added a feature that might work for you. It's a
crude export function on the Search Letterbox page. Search
for your region, then click the link at the bottom and
follow the directions: "Export Box Info - rudimentary
export; select page contents, paste in notepad, and save
as .csv." You can then open the .csv file in Excel.

I there are any PHP programmers out there that want to add
the tricky mime/type code to let this file save as a .csv
files without the nasty copy/paste, please contact me
offline. Otherwise, I'll research it someday and finish
the feature.

Choi
