just so you know:
To unsubscribe, send a message to:
letterbox-usa-unsubscribe@yahoogroups.com
That's all there is to it.

There are other options to receiving each e-mail sent to the list
individually. You can set yourself up to receive ONE daily digest of
all the e-mails sent that day, or you can set your account up to read
all the e-mail on the web whenever you want (In this case you receive
no e-mail from the list at all)
