From: Rick Simpson (simpson.rick@gmail.com) | Date: 2005-05-03 10:17:32 UTC-07:00
You need to rename your newletter.htm file to index.htm. You always
need your main page to be index.htm at geocities. That should fix
your main problem.

You need to use the img tag to insert an image on a page, rather than
link to the jpg. There's a lot of free online HTML tutorial that can
walk you thorugh it.

Hope that helps.
