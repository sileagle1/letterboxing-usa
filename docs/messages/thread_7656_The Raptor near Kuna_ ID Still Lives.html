The Raptor near Kuna, ID Still Lives

From: Aryn Davis (jedaryn02@yahoo.com) | Date: 2002-09-12 15:53:28 UTC-07:00
Greetings fellow letterboxes.
I wanted to let you all know that the Raptor Letterbox
near Kuna, ID is alive. When I stamped in today the
last folks to have been there were 7 Sep 01...a long
time ago. I guess everyone will have to start coming
to Idaho to visit and plant some more boxes and find
the few we have so far. Are there any other
Letterboxers out there who live in Idaho? I'm trying
to place some more, but it'd be more fun if others
joined in. =-)

If you do go to the one in Kuna I recommend you bring
a glove or long sleeve shirt. I was pretty freaked
out reaching in for it. I think the local prairie
dogs/rabbits played with it some. =-) I even saw a
Prairie Falcon flying by after I stamped in. Neat-O!!

See you on the trail.
Aryn from Eagle, ID.
