From: kayaktom2002 (kayakers@cox.net) | Date: 2003-12-29 00:03:36 UTC
Just an FYI - we found "Great Pond" box today (Simsbury); however,
it was VERY wet! Basically swimming in water. We dried it out, best
as we could. Pages in journal were still readable, but very damp.
We logged in and put the whole container and contents back in
a "drier spot". There is a large fallen limb, which touches the tree
at which the base it was initially hidden (currently in water). This
limb has a hollowed out knot, in which the box now resides.

We also found the DEP Massacoe State Forest box; as well as
the "Three Furry Friends". A great day, warm winter weather, for
boxing!
