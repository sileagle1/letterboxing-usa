Wendell State Forest, MA News

From: (bsennott@crocker.com) | Date: 2001-05-15 20:21:59 UTC
Hi everyone. Since 1998 I've had three letterboxes hidden at Wendell
State Forest in western Massachusetts.

The clues are here:

http://www.crocker.com/~bsennott/wendell.html

I'm planning on retiring those three boxes and replacing them with a
new box (or boxes) somewhere else in Wendell State Forest. The clues
for the new box will go on the same Web page, so no work is required
for the Webmasters (as far as I can tell).

I anticipate gathering up the current 3 boxes and hiding the new one(s)
before the end of May. So, if you were planning on a visit to Wendell
State Forest, check the Web page before you go for up-to-date clues. As
soon as I've hidden the new box, I'll post the clues (most likely the
same day).

This means that the three current Wendell boxes will soon have
"collector's item" status!

Bonnie

P.S. I plan to revisit my other Massachusetts boxes in early summer and
replace notepads or containers as needed. If anyone has news about the
condition/existence of the North Quabbin Reservoir boxes or North Mount
Sugarloaf, let me know.
