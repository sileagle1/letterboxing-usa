Good morning all,

Having just attended two gatherings (my second and third) with plans
to attend the Cape Caper in a couple weeks, I've stumbled upon the
following letterboxing dilemma...whether to count all the HHs lying
around on the tables at these events as "F" or not.

I know there are no hard and fast rules but I was just wondering
what the general consensus is on this. After all, no real effort is
expended in searching out these stamps. The numbers really mean
nothing compared with the simple joy of getting outdoors and
exercising the old grey matter but it'd be nice just to know how
others are arriving at their counts.

Thoughts?

I call it as it is labeled - If it is an Event Stamp (like the
Pacific NorthWest and KMartha), I count it as an E. Ditto with
the "official" Event stamp for that day.

If it is a HH, I count it as an HH.

If you have to ask some bizarre question of someone to get a stamp,
I'd call it as an F.

If you stumble across a box of tofu with a stamp inside, I'd call it
as an F.

If it is a personal traveler specific to one letterboxer, I'd call it
as an X.

I used to track PFX & Es--now, for the sake of ease, I've gotten
back to simply tracking PFX!

P for any permanent/hitchhiker/event letterboxes we've placed,
F for any letterboxes we find (permanent, hitchhikers,personal
travelers, virtual & event), and
X for any signature stamp we've exchanged with fellow letterboxers.

It really makes life quite easy this way!

I'll add my two cents here as I am sure that other will as well.

My counting is fairly simple to me.

Places = We have actually placed that box.

Finds = We have searched for and found that box.

Adopted = We have adopted another box that someone else planted.

Event = Actually event stamps, carved for an event, that we got while at the
event or that someone else has brought to an event. If we find it in the
wild, then it is a Find.

eXchanges = We have exchanged stamps with another letterboxer.

Traveler - An actual traveling box. Not Hitchhikers.

Hitchhiker - A actual hitchhiker that we have found or that was at an event.
If someone else is carrying the hitchhiker and allows us to stamp in, we
then count it as a HH.


But that's how we do it, not how it must be done, "Your mileage may vary."
