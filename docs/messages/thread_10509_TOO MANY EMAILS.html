If you enjoy this letterboxing group, but don't want all the emails---
CLICK on the "EDIT MY MEMBERSHIP" link toward the top right of the
letterbox group page and scroll down to the "MESSAGE DELIVERY"
section, choose the last option "NO EMAIL-- I'll view the messages on
the list". That should eliminate the problem---the messages don't go
to my email account, and that is how I have it set up--This is a
great list that is VERY active, I can see how it could clog up your
email!
