Compass usage

From: mulbie (mulbie@yahoo.com) | Date: 2002-05-01 12:28:02 UTC
Quite a few of the letterbox clues that we've followed require a
compass. Some others use "headings", somewhat more like a relative
bearing if you ask me. The idea seems to be to make the clues a
little vague so that it won't be too easy to find a box, and you might
have to try both the compass bearing and heading approach before
locating a treasure.
