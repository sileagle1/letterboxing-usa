I have been finding my 1.2 lt lock n locks at my local walmart. Yesterday I was lucky enough to find packs of smaller ones at a bigger Walmart! I was so excited! This pack has a round 100 ml container (2 3/4in), a rectangular 180 ml (2.75 x 3.5) and a square 600 ml (5.5 x 5.5 x 1.5) container. These are absolutely perfect sizes for some boxes I have planned. Best part was that the set was only $5. :) I've been paying $2.50-3.00 each for the others.

Question: does anyone else cry when they have to walk away from a huge stack of lock n locks? Good thing I hadn't cashed my check yet or I would have bought every last set they had! lol
