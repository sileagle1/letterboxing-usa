From: StarSaels (steves_1701@yahoo.com) | Date: 2008-07-02 20:38:35 UTC
I went looking for that not too long ago and found that you could get
the number of *listings* in a state and expand upon that...

Click on Search for Boxes, select a state, click Search.
You get the non-Mystery listings for that state.
Dump that to a .CSV file.
Open that in Excel.
Set the first "clean" cell in column H to equal the SUM of all the
H-column cells above it, and you have the number of BOXES.
