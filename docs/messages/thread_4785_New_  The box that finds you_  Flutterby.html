New! The box that finds you: Flutterby

From: leger de maine (legerdemaine@hotmail.com) | Date: 2002-03-26 20:58:05 UTC-05:00
"To invent is to think slightly askew."
 
     Introducing a new box variant: the postal box ...a concept is based on the premise that there is a universal joy in receiving something fun (and unexpected) in the mail. This traveler variation will move about via the mail. It will find you, and you, in turn, must find the address of another known letterboxer to send it on to. The best thing about this box (or box type) will be its ability to transcend the great distances which separate the various letterboxing communities. If recipients are conscientious about turning it around, it will also possess unique quality of being able to cover a lot of territory in a short amount of time.
 
     I've attached a pique at the stamp image, but the real interest will undoubtedly reside in the embellishments added by its future recipients. As someone noted a while back, it will be more fun to receive this later in the game. Can there be any question, though, as to where this one will flit first?
