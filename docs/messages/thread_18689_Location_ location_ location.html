I thought I'd speak my mind about something that I've thought could greatly improve and simplify these letterboxing communications! If people could please try to remember to include the name of the STATE (county would be great too) they're speaking of whenever they mention certain boxes and landmarks--creeks, mts, parks, etc. it would be wonderful!
