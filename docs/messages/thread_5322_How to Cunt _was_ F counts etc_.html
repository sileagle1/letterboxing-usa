Basically, I like to keep it simple. I don't really want to add E to
F, P, and X...or bother with a bunch of little details so here's what
I think:

I count Founds as all finds made the first time. I count Events in
with Founds (I found it...on the table) but I've only been to two
events, so it's not like it really throws the count much. I count
found hitchhikers here too -- which may have been what you found when
you happened on a box with two sets of stamps & logbooks. Regular box
maintenance or checking in on other's boxes when I'm in the area which
I visited before, don't count.

Placed are boxes my DH & I made and planted. I count hitchhikers we've
launched with these. BUT, if you need to replace a lost box, and you
change it's location etc. then I think it's allowable to add that as a
new placed. You are going through all the same efforts as placing a
new box, go ahead and count it. On the other hand if you need to
replace a box and you put it back exactly where it was, then you're
reusing old clues, stones, etc, that's only a half-effort, don't count
it.

Exchanged are those which are other people's personal stamps.

It's not like you reach god status when you reach a certain number,
and 1 or 2 this way or that isn't going to put you in prison. Count
however you think is right and don't sweat it.
