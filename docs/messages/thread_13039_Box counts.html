If you don't want to specify a number of boxes for your placements, you may use
-1.
This will remove the box from any calculations and counts, and will show "Not
specified" for the number of boxes when you view it.

Also, I've updated every box that was using 99 to do this, so the number of
boxes is much more reasonable.
We were at 4801 boxes after I did the update.
