Box Status Database Challenge

From: abacus00 (abacus00@yahoo.com) | Date: 2003-08-09 12:24:13 UTC
The initial test of the Box Status Database has been very
positive and I've received many favorable comments from
those who have used it so far. However, to take this to
the next level and make it a truely useful tool we need
more data. So I'm challenging each of you to take a few
minutes and enter a report for the last box or two you
have found or attempted (or more if you wish).

Why would you want to do this? With enough data, the
database will offer one spot where you can stop before
boxing to check box reports. You can sort on any of the
fields by clicking on the field name. You can search by
entering your search term in the box at the top right with
the button for some reason labeled "This Table". The data
can be imported into Excel for a file that can be used
off-line.

The fields are:
Box Name
State
City
Date
Finder
Status
Comments

To add a record:
Click on Database in the lefthand menu on the Letterbox-usa
Group website.
Click on the Box Status.
Click on the Add Record button.
Enter the data you wish to add.
Click on Add Record.

Please give it a try.
--Mahjong

Hi,

The database seems very well done. However, in my experience, there
is already a single place I check before I go boxing to see if
anything is wrong with any boxes...the main list itself. The search
feature makes it easy to search through all posts to find info you
want. Plus, it's nice to go to just one place for ALL boxing related
comments, narratives, opinions. It also seems easier when posting
narratives or status reports to simply create a new post and type in
your narrative rather than have to type in all the extra location
information about every box I find. I'd rather just talk
about "boxing in xyz area and these are the boxes I found and their
condition".

Just my two cents. It seems that the original light-hearted banter
about the TERM "sitrep" has turned into an issue with the actual
POSTING of narratives about letterboxing adventure. Most comments
I've seen indicate that folks enjoy such stories...
