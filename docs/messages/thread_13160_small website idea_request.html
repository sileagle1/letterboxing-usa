I was thinking it would be nice to have a "last updated" date field
(could be combined with the "Placed" date field or a completely
separate one) that would appear as another column in the index table
page for each region. This way, you could quickly scan down the index
page to be sure that the clues you may have already printed out
haven't been updated (possibly resulting in much frustration and a
failed hunt).

So, for example:
BOX NAME NUM BOXES UPDATED NEARBY CITY COUNTY
LetterboxZ 1 5/27/03 Yourtown Yourcounty
