From: Steve (boxdn@yahoo.com) | Date: 2005-05-22 12:43:44 UTC
The flags that appear on the list of clue names means the clue isn't
hosted on LbNA. If there isn't a flag then click on the clue and it
should have the flag in the top right hand corner with a list of
finders, if any. From the members page you can set it so that it
hides your boxes the ones you found and attemped/missing boxes. With
everything enabled you should only see clues for boxes you need to
get.

Boxdn
