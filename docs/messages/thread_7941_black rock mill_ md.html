HI

my husband and I found this box today and it is in good condition. However,
I just want to put out a little more information about the trail. It is
relatively flat, but it is a natural trail. My husband, who uses a
wheelchair could not have done this box with out helpers. I had to leave
him at one point and go on by myself - which made me a little nervous. We
encountered lots of roots and obstacles to our path like gullies, trees down
and large rocks. If you have small children this is not a stroller friendly
path. And at some points runs very close to the river. I would be very
careful with children along this path.

We decided to try this trail because the description said flat, narrow
trail. Although this was generally true - it is a very rustic path and it
would have been helpful to know this in advance - since we would not have
attempted this box. FYI - There are some of us out there who want to
participate in this hobby, but use alternative modes of transport. More
detailed information about the accessibility of the trail would be helpful.

Thanks
