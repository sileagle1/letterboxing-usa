I know it is counter intuitive but would it be helpful for the
LBNA 'new letterbox announcement' e-mail subjects to list State first
and then the county? Then when we sort our e-mails the states would
all sort together.

Instead of a sort by county, mixing up the states:

[lbox-announce] Fairfield County, CT
[lbox-announce] Monterey County, CA
[lbox-announce] Plymouth County, MA
[lbox-announce] Shasta County, CA
[lbox-announce] Tolland County, CT
[lbox-announce] Worchester County, MA

It would sort the States together:

[lbox-announce] CA, Monterey County
[lbox-announce] CA, Shasta County
[lbox-announce] CT, Fairfield County
[lbox-announce] CT, Tolland County
[lbox-announce] MA, Plymouth County
[lbox-announce] MA, Worchester County

It's just a suggestion. Many thanks for all the great work on the
site.
