Just a reminder to all

It is a nice gesture to add the state or some indicator in the subject as to
what state the discussion is centered on.

It makes it easy if you don't want to read about the area's away from your
home. You can easily skip the message without reading it

AND

not the state we might get all excited thinking we have a new box to find
only to be crushed when we find the box is 2000 miles away.

Please, no crushing!
