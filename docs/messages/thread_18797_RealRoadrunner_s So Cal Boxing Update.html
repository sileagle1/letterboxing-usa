Took advantage of a quick business trip to do a little boxing:

--Planted "The Transformed Man"--with a plea for help to the So Cal Boxing community, I

possibly a REALLY BAD location for a VERY GOOD stamp

picked up Santa's Travels HH as a nice surprise.

So, can't complain, really pleased with the results for a quick trip...my found count goes
up to 28, which is only 72 away from 100.
