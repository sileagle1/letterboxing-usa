It is up to you.

Go into www.groups.yahoo.com. Login. If you haven't set up a login and
password, do so. For an email address, use the exact same one that you used
to join letterbox-usa.

Once you are registered, you'll see letterbox-usa listed on the left of the
home page as one of your groups. Select the group, and you'll go our
Letterbox-USA home page.

On the right, is a column labeled membership. Select "Edit my membership".
The next page has a section called "Message Delivery". If you select "Daily
Digest" you'll get a digest once a day.

Have fun!


Pat & Wesley
