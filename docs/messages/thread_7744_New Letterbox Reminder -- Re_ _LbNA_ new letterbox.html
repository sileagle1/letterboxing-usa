Hi, Judi!

First, as a gentle reminder, remember to include the state abbreviation
in the Subject line for new clues -- this helps the various Webmasters
immensely!

Second, the Disclaimer is a normal part of the LbNA page where your clues
will go because it will actually reside in a "courtesy page" within the
LbNA's Web (we do this when posters don't have Web space of their own).

Third, remember to always include the following: (thanks, Judi, for
remembering these!)Name of the box, Nearest City, COUNTY, and State of
the box . This helps us determine the correct region of a state.

Great job with the clues!
