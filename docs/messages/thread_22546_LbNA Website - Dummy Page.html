I wanted to point out another thing about the dummy page
displayed for clues linked outside of LBNA. It normally
would just have the link for the off site clue, but if
you do like me and cut and paste the clues in LBNA in
addition to linking to an offsite clue page, those clues
will show up in addition to the link. This could be
confusing since these clues are not necessarily updated
like the linked clue page is, so be sure to always go
to the link for the most up to date clues. The reason
I do this is for backup, so if my clue pages get deleted
there will still be some clues available.
