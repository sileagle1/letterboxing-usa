The Davie Co. letterboxers were on the trail again this past Sunday. And after some confusion were able to eventually locate the Lake Daniel Park letterbox. Great stamp Tammy. A certain street name got us all turned around for a while. But once we found the swimming pool it all came together. Mimosa trees are easier to identify with foilage however. In the box there was a request to send postcards but no address. Help us out Tammy if you read this.
