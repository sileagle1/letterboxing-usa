Email a box's placers

From: Wes Garrison (wes@wesgarrison.us) | Date: 2003-05-19 02:10:11 UTC-05:00
I just added a feature.

You can now send a message to a box's placer by using the "Contact the
placer" link on the box's page.
The sender of the message never sees your email address, so don't worry
about spambots picking it up.

Note, this only works for boxes that don't redirect you to a personal site.
You'll have to find the contact info on the placer's page.

Comments appreciated.


Wes
