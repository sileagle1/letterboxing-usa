This morning, Peter J. McGuire, the Father of Labor Day, is honored with
his own Travelling Letterbox.

This letterbox has a separate Clue book, which is placed in a host box.
The Traveller is then hidden very nearby.
If you find the Clue, you must find the box. If you don't find the box,
then leave the clue in it's host for another.

Since Peter McGuire was never lonely, even when economically destitute,
we'd like it if he where not left alone too long. With that in mind, we
give you this hint: his first host is one of the Drew Clan's 300+ boxes.

Have a great Labor Day!!!
