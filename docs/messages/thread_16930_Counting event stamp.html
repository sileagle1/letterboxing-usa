Counting event stamp

From: theendgang (hurdhut@aol.com) | Date: 2003-11-21 05:57:03 UTC
We've been to 1 event. We counted the event stamp as just that, an
event stamp. Everyone wanted to sign our HH before we set it loose.
Being new, we allowed that. In the future I think we will keep HHs a
secret. I believe the event stamp should be counted, as it brings us
together to share this wonderful hobby. I don't believe, however,
that it should counted as a find. There should be a consensus at
events as to what the "official" event stamp is. Other stamps made
for that gathering should be planted so people can enjoy "The hunt".
Perhaps this issue should be the next poll that is taken.
