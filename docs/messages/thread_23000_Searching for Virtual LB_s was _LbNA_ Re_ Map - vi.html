If you are not joking, please excuse my assumption. For those who don't know about the
"search" function at LBNA, you can put word(s) into the search-in-clues area to bring up
boxes.

For example:

I am uncomfortable looking for boxes where "hungry bears" might be found. I'll type in
"hungry bears." If a box is listed, I read the clues. If I don't want to find this box, I will set
the box status to ignore. I could add "poisonous snakes, water mocassins, scorpions, etc."
to my list of boxes to skip. Nothing personal to the planters, but in a strange area, I don't
want to go missing or have little time to get medical attention.
