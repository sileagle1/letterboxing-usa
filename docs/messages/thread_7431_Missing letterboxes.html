Re: Re:[LbNA] Missing letterboxes

From: Ruthann Logsdon Zaroff (ruthann@ruthannzaroff.com) | Date: 2002-09-04 10:24:24 UTC-04:00
As one of the LbNA Webmasters I can say that we do try to make note of boxes
that are known missing. It is our policy, however, to state that a box is
missing only when the owner of the box reports it missing. Many times people
report that they couldn't find a box and say that it must be missing only to
have the owner revisit the box and it is exactly where it should have been.

Many boxers place boxes while on vacation, however, and revisiting is
impossible, so there certainly are boxes out there that go missing and are
not noted as so on the main LbNA page. We do try our best!

Ruthann
aka Mirkwood
