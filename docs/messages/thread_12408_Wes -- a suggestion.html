Wes,

Great work on the database site!

I don't know if it's been thought of or discussed, but it occurred to
me that if the site can now track who's logged in, and which boxes
belong to them, it could also track which boxes they've found.

I imagine it wouldn't be too difficult to add a checkbox next to
every listed box, and add buttons at the bottom of the page that
say "Show all unfound boxes," "Show all found boxes," and "Show all
boxes." The user checks off those boxes they've found, clicks "Show
all unfound boxes," and viola! they're presented with a list of boxes
they have yet to find. It would make it easy to take note of new
boxes, rather than having to search through the list trying to
remember which ones they've already found.

Just a thought!
