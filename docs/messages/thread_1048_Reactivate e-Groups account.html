Erik,

Most mailing list server programs have some sort of "bounce management"
program that deals with bad addresses. Essentially a threshold is set
(number of bounces in a given period of time), and once it is reached the
program unsubscribes you from the list and sends a message such as the
one you received. The problem that caused your address to bounce is most
likely to be related to your ISP's mail server having temporary
difficulty, but it could also be anywhere in between egroups and your
mail account. Generally, these problems go away if there is no real
problem with your address and you can just subscribe again using whatever
method the message told you to do. However if there are many others on
the list that got such a message, it would be more likely the problem had
something to do with the egroups mailing list server. Hope that helps.
